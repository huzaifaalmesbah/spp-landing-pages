Core Features

Two powerful tools.
One free plugin.

Smart Password Protect combines site-wide password protection and temporary magic-link access into a single, lightweight WordPress plugin.

Password Protection

Protect Entire Website

Lock down your entire WordPress site behind a password. Perfect for staging environments, maintenance mode, and private client previews.

  • One-click protection with a single password
  • IP whitelisting — automatic access bypass for trusted IPs
  • Role bypass — admins & logged-in users always get through
  • SEO-safe — automatic noindex headers when active
  • Customizable "Remember Me" duration — up to 7 days
  • Modern, responsive password form frontend
Temporary Login

Passwordless Temporary Access

Generate secure magic links for developers, clients, or teammates. No username, no password — links expire automatically, zero cleanup needed.

  • Self-expiring links — 1 hour, 1 day, 7 days, or custom date
  • No credentials shared — one-click login for the recipient
  • Assign WordPress roles — Administrator, Editor, Subscriber
  • Login usage limits & per-link usage tracking
  • Automatic logout when the link expires
  • Activity logs with IP address & browser details

Advanced Security Logs

Full audit trail, bot blocking, real-time token validation, and multi-layer auth checks.

19+ Languages

RTL-ready. Supports German, Arabic, French, Chinese, Hindi, Spanish and more.

60-Second Setup

Install directly from the WordPress dashboard. No coding, no config files needed.

How It Works

Up and running in minutes.

1

Install Plugin

Search "Smart Password Protect" in Plugins → Add New and click Install Now.

2

Set Your Password

Go to Smart Password Protect settings, enter a site password, and enable protection.

3

Generate Links

Create temporary magic links from the Temporary Login tab. Set role, expiry, and usage limits.

4

Share & Monitor

Share the link. Monitor usage in activity logs. Everything auto-cleans when expired.

Security First

Hardened against
every attack vector.

Smart Password Protect is built with zero-trust principles. Every access request is validated at multiple layers — temporary users can never abuse their access or disable the plugin.

View on WordPress.org
Self-Preservation Logic

Temporary users cannot deactivate, delete, or modify the plugin — ever.

Profile Lockdown

Temp users are blocked from editing profiles or managing other users.

Automatic Cleanup

Expired links and temporary accounts are removed automatically by WP Cron.

Secure HttpOnly Cookies

Auth cookies are HttpOnly and expire automatically — no session drift.

Bot Protection

Crawlers and bots are automatically identified and blocked from bypass attempts.

Multi-Layer Validation

Tokens are validated on every request — there are no gaps in the protection chain.
